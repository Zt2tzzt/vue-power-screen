# 三、充电桩数据分析组件编写

## 1.countup.js 库引入

引入 counteup.js 库，实现数据滚动动画。

## 2.数据请求并填充

使用 `nextTick` 将滚动动画放在下一次 DOM 更新完成后执行。

src\components\BottomPanel.vue

```js
import { nextTick, watch } from 'vue';
import { CountUp } from 'countup.js';

const props = defineProps({
	panelData: {
		type: Array,
		default: function() {
			return []
		}
	}
})

watch(() => props.panelData, newVal => {
	nextTick(() => {
		newVal.forEach(item => {
			new CountUp(`total-num-${item.id}`, item.totalNum, {
				decimalPlaces: 1, // 保留 1 位小数
				duration: 2,
				useGrouping: false // 1000 -> 1,000
			}).start()
			new CountUp(`percentage-num-${item.id}`, item.percentage, {
				decimalPlaces: 1,
				duration: 2,
				useGrouping: false,
				suffix: '%'
			}).start()
		})
	})
})
```

## 2.中间组件入场动画

整个 svg 元素入场动画

“房子”的动画

src\components\CenterSvg.vue

```js
import { onMounted } from 'vue';
import gsap from 'gsap'

onMounted(() => {
	const timeline = gsap.timeline()

  // 整个 svg 元素入场动画
	timeline.fromTo(['#dongxiao'], {
		scale: 0.2,
		y: 50
	}, {
		duration: 1,
		scale: 1,
		y: 0
	})
  // “房子”的动画
	.fromTo(['#center-house'], {
		scale: 0.6,
		transformOrigin: 'bottom',
		opacity: 0.2,
		y: 30
	}, {
		duration: 1,
		scale: 1,
		transformOrigin: 'bottom',
		opacity: 1,
		y: 0
	}, '-=1') // '-=1'，代表时间线中的该动画，与上一个动画同时执行
})
```
